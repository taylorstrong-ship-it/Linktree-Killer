# Layer 2 & 3: Sound Files

This folder should contain your UI sound effects:

- `pop.mp3` - Light click/tap sound
- `click.mp3` - Button click sound  
- `success.mp3` - Success action sound
- `error.mp3` - Error notification sound

## Where to get sounds:
1. **Freesound.org** - Free UI sound effects
2. **Zapsplat.com** - Free sound library
3. **Generate with AI** - Use tools like ElevenLabs or similar

## Requirements:
- Keep files TINY (under 10kb each)
- Use MP3 format for best browser compatibility
- Keep volume subtle (the code sets it to 0.2-0.3)

## To add a sound:
1. Download/generate your sound file
2. Save it to this folder (e.g., `pop.mp3`)
3. It's automatically available via the Sound utility!
